# datascience_project
Data Science Project Engim

Progetto di Data Science nell'ambito del corso IFTS in Tecniche di Gestione e Progettazione dei Database.
Il progetto è oggetto di prova per la materia "Data Science".

Attraverso un jupyter notebook, ho condotto un'analisi in python di un dataset riguardante l'accesso all'elettricità nel mondo.
Ho esplorato il dataset, l'ho pulito e integrato con altri dati, e ho elaborato grafici di sintesi.
